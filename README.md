# HTML Portfolio

## Overview

This project is my first personal portfolio website built using only HTML. It introduces who I am, my journey into web development, my career goals, and the projects I will build as I continue learning.

The purpose of this project is to practice writing semantic HTML, organizing a website into multiple pages, and building a strong foundation before learning CSS and JavaScript.

---

## Features

- Navigation menu
- About Me page
- My Journey page
- Developer Goals page
- Projects page
- Contact page
- Semantic HTML structure
- Responsive viewport configuration
